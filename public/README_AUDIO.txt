PETUNJUK PENEMPATAN AUDIO:
1. Siapkan file musik Anda dalam format .mp3 (misalnya lagu Jamrud - Selamat Ulang Tahun).
2. Ubah nama file tersebut menjadi: jamrud_selamat_ulang_tahun.mp3
3. Salin/pindahkan file tersebut ke dalam folder: public/
4. Setelah file ditempatkan di public/jamrud_selamat_ulang_tahun.mp3, website akan memutar lagu tersebut secara otomatis saat tombol "Buka Kado" diklik.

Catatan Fallback:
Jika file ini tidak ditemukan atau gagal dimuat, website memiliki sistem cadangan (fallback) otomatis berbasis Web Audio API yang akan mensintesis melodi "Happy Birthday" menggunakan efek musik box (chime oscillator) yang elegan, sehingga website tidak akan pernah hening.
